# Sahih al-Bukhari JSON data

Source candidate: https://github.com/4thel00z/hadith.json/tree/master/bukhari

This folder is prepared for importing Sahih al-Bukhari JSON files into Sahihayn.

Source notes:
- The source repository says its hadith JSON files were scraped from Sunnah.com.
- The Bukhari folder contains files such as `bukhari_1.json`, `bukhari_6.json`, `bukhari_14.json`, etc.
- `bukhari_1.json` contains references like `Sahih al-Bukhari 1`, Arabic text and English translation.
